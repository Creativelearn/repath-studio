## How to Run

Install the dependencies.
```
npm install
```
Build and watch the project files . 
```
npm run dev
```
Run electron on a different terminal.
```
npm run electron
```

## Useful shortcuts

```
Ctrl+Shift+I Toggle devtools
Ctrl+H Toggle 10x
Ctrl+R Reload app
```
